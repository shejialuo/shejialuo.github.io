Telegram代收QQ和微信消息

shejialuo
2022-11-07 (Updated: 2022-11-07)
工具, 技术
  1. 前言
    在ArchLinux i3上想用QQ和微信别人交流，是一件极其困难的事情，腾讯开发的linux for qq总是经常的出现段错误，而且界面过于丑陋。同时，腾讯为UOS开发的WeChat仅仅通过Electron封装了网页端，聊天记录都不能保存。故尝试采用Telegram代收QQ和微信消息。首先本文感谢以下的开源项目：

efb-qq-slave
efb-wechat-slave
enForwarderBot
mirai
mirai-api-http
mirai-console-loader
其中，efb-qq-slave的作者发表了博客安装并使用EFB：在Telegram收发QQ消息，但由于coolQ在2020年8月2日的关闭（详见知乎）该博文在某些方面失去了时效性。efb-qq-slave的作者奶冰大佬最近重构了项目，本篇博客拟记录部署服务的过程，同时本博客采用了该博文的部分内容，本博客所用的博文均遵循CC BY-NC-SA 4.0 许可协议，符合该博文的采用的CC BY-NC-SA 4.0 许可协议。

  1. 前提条件
    在开始部署之前，需要完成以下的准备工作：

墙外的一台VPS
一台电脑，Windows系统采用Windows Terminal和PowerShell7进行SSH连接
3. 环境搭建
由于我的服务器环境为Ubuntu 20.04 LTS，本文按照该操作系统进行描述。如果你采用的是其他发行版的Linux服务器，其过程并无本质区别。由于依赖或LTS版本的问题，你可能会遇到很多坑，如果你熟悉Docker，建议采取Docker部署服务，同时也为你的服务器中的其他服务提供了隔离。

3.1 安装Python与非Python依赖
对于非Python依赖，直接通过包管理器安装：

sudo apt-get install libopus0 ffmpeg libmagic1 git libssl-dev
显然，使用Docker你就能避免以上的烦恼。

3.2 从PyPI下载相关软件
pip3 install ehforwarderbot
pip3 install -U git+https://github.com/milkice233/efb-qq-slave
pip3 install efb-telegram-master
pip3 install efb-wechat-slave
4. 配置
EFB的工作机制是一个唯一的主端和多个从端进行通信，由于需要同时代收QQ和微信，先创建配置文件夹：

mkdir -p /.ehforwarderbot/profiles/qq/
mkdir -p ~/.ehforwarderbot/profiles/qq/blueset.telegram
mkdir -p ~/.ehforwarderbot/profiles/qq/milkice.qq
mkdir -p ~/.ehforwarderbot/profiles/wechat/
mkdir -p ~/.ehforwarderbot/profiles/wechat/blueset.telegram
mkdir -p ~/.ehforwarderbot/profiles/wechat/blueset.wechat
在/.ehforwarderbot/profiles/qq/创建文件config.yaml，内容如下：

master_channel: blueset.telegram
slave_channels:

  • milkice.qq
    在~/.ehforwarderbot/profiles/wechat创建文件config.yaml, 内容如下：

master_channel: blueset.telegram
slave_channels:

  • blueset.wechat
    4.1 配置主端
    本部分请阅读安装并使用EFB：在Telegram收发QQ消息中的配置主端部分。

其中，/setcommands部分应采取如下：

help - Show commands list.
link - Link a remote chat to a group.
unlink_all - Unlink all remote chats from a group.
info - Display information of the current Telegram chat.
chat - Generate a chat head.
extra - Access additional features from Slave Channels.
update_info - Update info of linked Telegram group.
react - Send a reaction to a message, or show a list of reactors.
rm - Remove a message from its remote chat.
同理，对于微信主端再新建一个Telegram的Bot，操作同上。

4.2 创建ETM配置文件
本部分请阅读安装并使用EFB：在Telegram收发QQ消息中的创建ETM配置文件。微信同理。

4.3 配置从端
本部分涉及到efb-qq-slave采用的QQ客户端为Mirai模式，参考GitHub官方文档。

这里会涉及到了一个问题，滑块验证，Mirai的控制台会提供足够的信息使得你解决这个问题。

对于微信而言，efb-wechat-slave已经内置了微信客户端，不需要折腾。

    输入ehforwarderbot –profile qq启动
    5.2 微信
    输入ehforwarderbot –profile wechat启动
